# ATM Interface - Java

This is a console-based ATM Interface application developed using Java as part of my internship at Oasis Infobyte.

## Features
- User authentication using User ID and PIN
- Transaction history
- Withdraw money
- Deposit money
- Transfer money
- Quit option

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- VS Code

## How to Run
1. Compile the program:
   javac *.java
2. Run the program:
   java Main
